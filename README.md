# 电力市场书籍集合

本仓库收录电力市场相关书籍的 GitBook 格式整理版本。

## 书籍列表

### [电力现货市场101问](book1/)

- **编者：** 国家电力调度控制中心
- **出版社：** 中国电力出版社（2021年10月）
- **简介：** 以问答形式讲述电力现货市场原理性问题和国内现货市场建设普遍关注的特有问题，共101问
- **章节：** 3章（电力现货市场基础、国内电力现货市场建设关键问题、技术支持系统）

### [电力市场化交易必读](book2/)

- **编者：** 陈向群、罗朝春 等
- **出版社：** 中国电力出版社（2023年6月）
- **ISBN：** 978-7-5198-7681-4
- **简介：** 全面介绍电力市场基本概念和运行实操，涵盖中长期交易、现货交易、结算、合同管理、可再生能源消纳等核心业务
- **章节：** 10章（从电力市场概述到电力交易综合管理）

### [电力交易工作问答](book3/)

- **编者：** 陈向群、罗朝春、成涛、刘勇、徐文超、葛亮 等
- **出版社：** 中国电力出版社（2021年10月，2022年4月第三次印刷）
- **ISBN：** 978-7-5198-5985-5
- **简介：** 以电力市场操作技能为主线，系统回答电力交易工作中经常遇到的问题，突出实用性、针对性、先进性和严谨性
- **章节：** 12章（从概述到综合部分，共432问）
